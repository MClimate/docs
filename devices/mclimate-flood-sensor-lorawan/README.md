## Features

* Flood detection
* Device tampering detection (opening the device)
* Temperature sensor
* Buzzer
* LED

## Power supply

* Battery type: CR123A
* Operating voltage: 2.5 - 3.6 VDC
* Expected battery life: Up to 10 years (depending on configuration and environment)
