## General information <a href="#general-information" id="general-information"></a>

MClimate Open/Close Sensor LoRaWANÂ® is a device detecting the event of opening or closing windows, doors, cabinets and more. It features long battery life, LED, temperature sensor and a button that triggers an uplink. The device sends an uplink for every event of opening/closing and keeps an internal counter of the total number of events.

## Features <a href="#features" id="features"></a>

* Magnetic reed switch
* Button
* Temperature sensor

## Power supply <a href="#power-supply" id="power-supply"></a>

* Battery type: 1/2 AA or ER14250
* Operating voltage: 2.5 - 3.6VDC
* Expected battery life: Up to 10 years (depending on configuration and environment)
* _**Device does not operate with rechargeable batteries!**_

## Compatibility

* LoRaWAN 1.0.3, class A device, EU868
* Encryption: LoRaWAN End-to-end encryption (AES-CTR)
* Activation: OTAA
* Link budget: 130dB
* RF Transmit Power: 14dB
